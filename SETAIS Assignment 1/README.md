# Running the tests

Here we explain how to run the tests for both subgroups.

## Subgroup 1
Run the following command in the terminal to run the metamorph tests:

````bash
python tests_group1/Metamorph.py
````

and the following command to run the partition tests:
````bash
python tests_group1/Partition.py
````

## Subgroup 2

Run the following command in the terminal to run all tests:

````bash
python tests_group2/group2_tests.py
````
